
# 🌍 AI Language Translator

## About This Project

As a first-year B.Tech student in AI & Data Science, I wanted to build a practical AI application that could solve a real-world communication problem.

This project is an AI-powered Language Translator developed using Python and Streamlit. It allows users to translate text between multiple languages instantly, listen to the translated output using text-to-speech technology, and easily copy the translated text for further use.

Through this project, I explored how Natural Language Processing (NLP) tools can be integrated into user-friendly applications and gained hands-on experience in developing interactive AI-powered software.

##  Project Workflow

1. User enters text
2. User selects the target language
3. Application translates the text
4. Translated text is displayed instantly
5. User can copy the translated text
6. User can listen to the translated output using Text-to-Speech
